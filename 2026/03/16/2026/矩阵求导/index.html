矩阵求导

在机器学习中不乏有涉及矩阵的运算，其中矩阵求导更是涉及优化问题的关键。虽然现在有很多开包即用的api可以帮助我们完成这个工作，但是我还是想了解一些它们底层的工作原理。用这篇帖子记录之。

标量函数对向量的求导

在高等数学中我们学习了多元函数的导数，例如$f(y_1,y_2)$是一个对$y_1,y_2$的函数，其导数如下

$\frac{\partial f(y_1,y_2)}{\partial y_1},\frac{\partial f(y_1,y_2)}{\partial y_2}$

基于此我们可以定义标量函数对向量求导，因为多元函数中“多元”的变量本身就可以看作一个向量，而最终的向量的函数是一个标量

定义如下：

  1. 有向量 $\vec{y} = (y_1,y_2,…,y_m)^T$ 作为变量
  2. 函数 $f(\vec{y})$ 对变量 $\vec{y}$ 对导数为 $\frac{df(\vec{y})}{d\vec{y}}$
  3. $\frac{df(\vec{y})}{d\vec{y}} = (\frac{\partial f(\vec{y})}{\partial y_1},\frac{\partial f(\vec{y})}{\partial y_2},…,\frac{\partial f(\vec{y})}{\partial y_m})^T$

注意：

  1. 显然，导数作为一个向量，其中的分量就是我们刚刚多元函数求导时的各个偏导；
  2. 注意导数的向量此时带有一个转置，是一个列向量和分母$\vec{y}$相同，此时称为分母布局，因为行数与分母（$\vec{y}）$相同；
  3. 导数也可以不带转置，此时称为分子布局，因为行数与分子相同（分子是标量（$f(\vec{y})$），可以看作行数为1的向量）；
  4. 分母布局与分子布局互为转置；

一个具体的例子:

  1. $\vec{y} = (y_1,y_2)^T,f(\vec{y}) = y_1^2+y_2^2$
  2. $\frac{df(\vec{y})}{d\vec{y}} = (2y_1,2y_2)^T$

向量函数对向量的求导

在（向量的）标量函数对向量求导的基础上，向量函数本质上是多个标量函数组成的向量，所以向量函数对自变量（向量）的求导实际上就是 向量函数中 作为分量的每个标量函数 对于自变量（向量）的求导

定义如下

  1. 有向量函数 $\vec{f}(\vec{y}) = (f_1(\vec{y}),f_2(\vec{y}),…,f_n(\vec{y}))^T$，注意，这是一个n行的向量；
  2. 自变量 $\vec{y} = (y_1,y_2,…y_m)^T$，注意这是一个m行的向量
  3. 上述函数对自变量的导数记做 $\frac{\partial \vec{f}(\vec{y})}{\partial\vec{y}}$
  4. $\frac{\partial \vec{f}(\vec{y})}{\partial\vec{y}} = (\frac{\vec{f}(\vec{y})}{y_1},\frac{\vec{f}(\vec{y})}{y_2},…,\frac{\vec{f}(\vec{y})}{y_m})^T$，然后按照刚才的标量函数（当前向量函数的各个分量），在列上进行展开，得到下述矩阵

$
\begin{pmatrix}
\frac{\partial f_1(\vec{y})}{y_1}&\cdots&\frac{\partial f_n(\vec{y})}{y_1}\
\vdots&\ddots&\vdots\
\frac{\partial f_1(\vec{y})}{\partial y_m}&\cdots&\frac{\partial f_n(\vec{y})}{y_m}
\end{pmatrix}
$

显然上述是分母布局，因为矩阵的行数为$m$，和分母的自变量$\vec{y}$相同

一个具体的例子

  1. $\vec{f}(\vec{y}) = (f_1(\vec{y}),f_2(\vec{y}))^T = (y_1^2+y_2^2+y_3,y_3^2+2y_1)^T$
  2. $\vec{y} = (y_1,y_2,y_3)^T$
  3. $\frac{\partial \vec{f}(\vec{y})}{\partial \vec{y}}$ 展开如下（分母布局）

$
\begin{pmatrix}
2y_1&2\
2y_2&0\
1&2y_3
\end{pmatrix}
$

常用的结论

  1. 如果有$\vec{y} = (y_1,y_2,…,y_m)^T$
  2. 其中A是一个矩阵，展开如下

$
\begin{pmatrix}
a_{11}\cdots a_{1m}\
\vdots \ddots \vdots\
a_{m1}\cdots a_{mm}
\end{pmatrix}
$

那么有如下两个结论

  1. $\frac{\partial A\vec{y}}{\vec{y}} = A^T$
  2. $\frac{\partial\vec{y}^T A \vec{y}}{\vec{y}} = A\vec{y}+A^T\vec{y}$，当$A^T = A$（$A$是对称的）时，导数等于$2A\vec{y}$

在这样的结论下，一般的线性回归使用矩阵建模，求解最小二乘法会非常简单！这里不再展开赘述，可以查看如下视频bilibili

（但是要注意，矩阵的逆不一定有解，所以可能无法得到线性回归的解析解，但有很多方法可以得到几何解）

一种很好的理解：通过矩阵求导，我们可以将一个最优化问题转换为求解矩阵方程，而在矩阵方程的层面考虑问题，或许会看到很多能够简化的地方…

矩阵求导的链式法则

不失一般性考虑一个外层是标量函数，内层是向量函数的嵌套函数

  1. $J = f(\vec{y}(\vec{u}))$
  2. $\frac{\partial J}{\partial \vec{u}} = \frac{\partial\vec{y}}{\partial\vec{u}}\frac{\partial J}{\partial \vec{y}}$

注意，链式展开的时候从右往左写，这样矩阵相乘的维度才能够对得上
