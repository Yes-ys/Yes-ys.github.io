Pytorch的自动求导机制

反向传播时对于计算图上各种函数的处理

  1. If the function is differentiable and thus a gradient exists at the current point, use it.
  2. If the function is convex (at least locally), use the sub-gradient of minimum norm.
  • 补充sub-gradient（次梯度）：
  • 使用场景，主要用于凸函数（Convex Function）。 在凸函数中，有些点可能是“尖角”（不可导），比如$f(x) = |x|,x=0$处。虽然这里没有传统意义上的切线（导数），但我们可以画出无数条在函数图像“下方”且通过该点的直线。
  • 数学定义：假设$f$是一个凸函数。对于定义域内的某一点$x_0$，如果存在一个向量$g$，使得对于定义域内的任意$x$，都满足以下不等式：$f(x)\ge f(x_0)+g^T(x-x_0)$那么，$g$就被称为函数$f$在$x_0$处的次梯度。
  • 为什么pytorch中使用最小范数？在机器学习的自动求导代码实现中，当遇到不可导的点时，程序不能返回一个集合（比如$[-1,1]$，它必须返回一个具体的数值给反向传播去更新权重。既然次微分是一个集合（可能包含无穷多个合法的次梯度），选哪一个最好呢？在梯度下降中，梯度指向函数值增长最快的方向，负梯度指向下降最快的方向。在不可导点，选择范数最小的次梯度意味着我们在做最“保守”且最平稳的更新，有助于优化算法（如 SGD）收敛，避免在不可导的“尖角”处发生剧烈震荡。数学和优化理论上的标准做法是：选择次微分集合中，范数（绝对值或长度）最小的那个次梯度（sub-gradient of minimum norm）。
  1. If the function is concave (at least locally), use the super-gradient of minimum norm (consider -f(x) and apply the previous point).
  • 至于super-gradient可以相对应的理解，对于凹函数，超梯度就是让得到的 直线/平面/超平面 在对应点的$f(x_0)+g^T(x-x_0)$始终大于$f(x)$的向量
  1. If the function is defined, define the gradient at the current point by continuity (note that inf is possible here, for example for sqrt(0)). If multiple values are possible, pick one arbitrarily.
  2. If the function is not defined (sqrt(-1), log(-1) or most functions when the input is NaN, for example) then the value used as the gradient is arbitrary (we might also raise an error but that is not guaranteed). Most functions will use NaN as the gradient, but for performance reasons, some functions will use other values (log(-1), for example).
  3. If the function is not a deterministic mapping (i.e. it is not a mathematical function), it will be marked as non-differentiable. This will make it error out in the backward if used on tensors that require grad outside of a no_grad environment.
